# MindEase – Mental Health Chatbot

MindEase is a simple web-based chatbot designed to provide supportive
mental health–focused responses using the OpenAI ChatGPT API.

## Features
- Conversational interface for mental health support
- Uses prompt-based control to guide responses
- Simple UI built with basic CSS
- Backend integration with OpenAI API

## Tech Stack
- JavaScript
- OpenAI API
- HTML, CSS
- Node.js

## Purpose
This project was built to explore API integration, prompt-based response
control, and basic chatbot interfaces.
